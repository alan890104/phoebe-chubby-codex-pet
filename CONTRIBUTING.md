# Come research the chubby

Bug reports, better wording, installation fixes, and reliable attribution leads are welcome.

Please keep contributions:

- kind and playful;
- non-commercial;
- free of copied third-party sticker packs or unverified ownership claims;
- compatible with the Codex v2 pet format.

Before opening a pull request, run:

```bash
python3 scripts/build_previews.py
python3 scripts/validate_release.py
bash -n scripts/install.sh scripts/build_release.sh
```

For a visual bug, include the action name and a screenshot or short recording. For a provenance correction, link the earliest reliable primary source you have.
