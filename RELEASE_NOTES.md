# 菲比啾比 × Phoebe Chubby v2.0.0

把會跑、會跳、也會生氣的菲比啾比安裝到 Codex。這是使用 GPT Image 2 製作圖像的非官方、非商業迷因寵物專案。

Add the running, jumping, occasionally angry Phoebe Chubby to Codex. This is an unofficial, non-commercial meme-pet project using images generated with GPT Image 2.

## Highlights / 本次內容

- Sprite atlas with 73 used cells
- GIF preview for every action and the 16-direction look loop
- Every preview rebuilt from the cleaned final atlas — no chroma-green outline
- Correct seated active-processing `running` loop with attached arms and single O mouths
- Traditional Chinese, Simplified Chinese, and English guides
- One-command install and in-place update on macOS, Linux, and Windows
- Automatic English, Traditional Chinese, and Simplified Chinese pet names and descriptions
- Stable `phoebe-chubby` ID with backups kept outside the Pets scan folder, preventing duplicate pets
- Image-rich GitHub Pages site in English, Traditional Chinese, and Simplified Chinese
- One-click Codex install button with a command-line fallback when browsers block app links
- Checksums and repeatable release validation

安裝後到 Codex Settings → Pets → Refresh，選擇「菲比啾比」即可。啾比！
